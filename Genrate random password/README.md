## Overview

This project generates a strong 8-character random password with a single click.  
Every generated password includes:

- Uppercase letters  
- Lowercase letters  
- Numbers  
- Special characters  

The UI includes a password display box, a copy-to-clipboard icon, and a generate button for quick access.

## Features

- **Strong Password Generation:** Creates an 8-character password containing all character types.  
- **One-Click Generate:** Instantly produce a new secure password.  
- **Copy to Clipboard:** Easily copy the generated password.  
- **Simple & Clean UI:** Modern, responsive interface.  
- **Randomization:** Uses JavaScript `Math.random()` for strong randomness.
