# Currency Converter App

The **Currency Converter App** is a simple and responsive web application built using **HTML, CSS, and JavaScript**.  
It allows users to convert currencies in real time using live exchange rates and displays country flags for better user experience.

## Overview

This project helps users convert one currency to another by selecting currencies from dropdown lists.  
It dynamically fetches real-time exchange rates and updates country flags based on selected currencies.

The app is beginner-friendly and demonstrates:
- API integration
- DOM manipulation
- Event handling
- Dynamic UI updates

## Features

- **Real-time currency conversion**
- **Supports multiple international currencies**
- **Automatic country flag update**
- **Swap currencies with one click**
- **Default currency selection**
- **Responsive and clean UI**
- **Error handling for invalid input**

## How It Works

1. User enters an amount.
2. Selects source and target currencies.
3. Clicks the **Get Exchange Rate** button.
4. JavaScript fetches live exchange data from the API.
5. Converted amount is displayed instantly.
6. Users can swap currencies using the exchange icon.

## Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Logic and API handling
- **Font Awesome** – Icons
- **Flags API** – Country flags
