# Healthy Food Restaurant Landing Page

Welcome to the **Healthy Food Restaurant Landing Page** project! This is a modern, responsive, and visually engaging homepage for a healthy food restaurant. It promotes nutritious dining, showcases menu items, features testimonials, and includes an interactive contact section.

## Features

- **Navigation Menu:** Smooth, responsive menu with hamburger toggle for mobile devices.
- **Hero Section:** Eye-catching banner with a strong call-to-action.
- **About Section:** Introduces the restaurant and its mission.
- **Food Categories:** Educates users about types of healthy food (Fruits, Grains, Vegetables).
- **Food Menu:** Displays multiple menu items with images, descriptions, and pricing.
- **Testimonials:** Customer reviews with star ratings and photos.
- **Contact Section:** Contains contact form and restaurant image.
- **Footer:** Simple and informative with copyright.
- **Smooth Scrolling:** Enabled via jQuery for a polished user experience.
- **Responsive Design:** Optimized for all screen sizes (desktop, tablet, and mobile).
