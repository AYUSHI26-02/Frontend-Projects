# World Travel Landing Page

Welcome to the **World Travel Landing Page** project! This is a sleek, modern, and responsive homepage designed for a travel agency or tourism-focused business. It includes visually appealing content sections, a call-to-action, and interactive forms to engage potential clients and encourage bookings or inquiries.

## Overview

This landing page showcases travel destinations, services, and contact options for a fictional travel agency called **World Travel (WT)**. It highlights scenic travel spots, unique selling points, and makes it easy for users to get in touch via a responsive contact form.

## Features

- **Header Section:** Bold hero banner with a travel-themed background and prominent call-to-action.
- **Destinations Showcase:** Highlights featured locations (e.g., Gujrat, India and Busan, South Korea) with images and descriptions.
- **Feature Boxes:** Showcases key advantages like Adventure, Low Prices, and Experience.
- **About Page:** Details about the company with stats (e.g., places traveled, number of guides) and mission highlights.
- **Contact Page:** Includes contact information, a company image, and a responsive contact form.
- **Responsive Design:** Fully mobile-friendly and adapts to all screen sizes.
- **Font Awesome Integration:** Uses icons for visual enhancement and better user experience.

## Technologies Used

- **HTML5**
- **CSS3**
- **Font Awesome**
- **Google Fonts** (Montserrat)
