# Image Gallery App

The **Image Gallery App** is a modern and responsive web application built using **HTML, CSS, and JavaScript**.  
It allows users to search for high-quality images, view them in a gallery layout, preview images in a lightbox, and download images instantly using the **Pexels API**.

## Overview

This project fetches images dynamically from the **Pexels API** and displays them in a masonry-style gallery layout.  
Users can search for images by keywords, load more images, preview them in a lightbox, and download them directly.

The project demonstrates:
- API integration
- Dynamic DOM manipulation
- Event handling
- Responsive UI design

## Features

- **Search images by keyword**
- **Masonry-style image gallery**
- **Load more images button**
- **Lightbox image preview**
- **Photographer name display**
- **Download image feature**
- **Responsive and modern UI**

## How It Works

1. On page load, curated images are fetched from the Pexels API.
2. Users can enter a keyword in the search box and press Enter.
3. The gallery updates dynamically with searched images.
4. Clicking on an image opens a lightbox preview.
5. Users can download images directly from the lightbox or gallery.
6. The "Load More" button fetches additional images.

## Technologies Used

- **CSS3** – Styling and layout
