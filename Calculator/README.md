# JavaScript Calculator

Welcome to the JavaScript Calculator project!

## Overview

This project is a beginner-friendly calculator created using **basic JavaScript logic**.  
It includes essential arithmetic functions like addition, subtraction, multiplication, and division.  
The UI is designed with a modern, soft-shadow (neumorphism) look to make the calculator visually appealing and easy to use.

## Features

- **Simple & Clean UI:** Easy-to-use layout with responsive design.  
- **Basic Operations:** Supports +, –, ×, ÷.  
- **AC Button:** Clears the entire input.  
- **DE Button:** Deletes the last character entered.  
- **Decimal Support:** Allows decimal point calculations.  
- **Instant Calculation:** Uses JavaScript's `eval()` to compute results.  
- **Responsive Layout:** Works smoothly on desktop and mobile screens.
