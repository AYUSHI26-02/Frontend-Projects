# Quote Generator

The **Quote Generator** is a simple and elegant web application built using **HTML, CSS, and JavaScript**.  
It fetches random inspirational quotes from an external API and allows users to share their favorite quotes directly on **Twitter** with one click.

## Overview

This project displays a **Quote of the Day** with the author’s name.  
Users can generate a new quote instantly and share it on Twitter using the built-in tweet button.

The UI is minimal, modern, and responsive, making it suitable for both desktop and mobile users.

## Features

- **Random quote generation**
- **Displays quote author**
- **One-click Twitter sharing**
- **Modern and clean UI**
- **Responsive design**
- **Error handling for failed API requests**

## How It Works

1. On page load, a random quote is fetched from the API.
2. Clicking the **New Quote** button fetches a new quote.
3. Clicking the **Tweet** button opens Twitter with the quote pre-filled.
4. Quotes are fetched dynamically using JavaScript `fetch()`.
