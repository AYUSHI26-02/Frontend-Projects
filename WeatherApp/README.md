# Weatherly – Weather App

Weatherly is a modern and interactive **Weather Application** built using **HTML, CSS, and JavaScript**.  
It allows users to search for any city and view **current weather conditions** along with a **multi-day forecast**, enhanced with dynamic background videos based on weather conditions.

## Overview

Weatherly uses the **OpenWeatherMap API** to fetch real-time weather data.  
Based on the weather condition (clear, rain, snow, clouds), the app dynamically displays a relevant **background video**, creating an engaging user experience.

The UI is clean, responsive, and suitable for both desktop and mobile screens.

## Features

- **Search weather by city name**
- **Real-time temperature and weather details**
- **Feels-like temperature, humidity, and wind speed**
- **5-day weather forecast**
- **Dynamic background videos** based on weather conditions
- **Modern and responsive UI**
- **Error-safe icon handling**

## How It Works

1. User enters a city name in the input field.
2. JavaScript sends requests to:
   - Current weather API
   - Forecast API
3. Weather data is displayed dynamically on the page.
4. A background video is selected based on the main weather condition:
   - Clear
   - Rain
   - Snow
   - Clouds

## Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Logic and API handling
- **OpenWeatherMap API** – Weather data
- **Pexels Videos** – Background videos
