# QR Code Generator

This is a simple and fast **QR Code Generator** built using **HTML, CSS, and JavaScript**.  
It allows users to enter any text or URL and instantly convert it into a scannable QR code using a free API.  
The interface is clean, minimal, and user-friendly.

## Overview

The QR Code Generator takes user input and displays a QR image generated from the API:  
`https://api.qrserver.com/v1/create-qr-code/`  

It includes smooth UI animations, error handling, and a responsive layout.  
The project is ideal for beginners learning JavaScript DOM manipulation and API usage.

## Features

- **Real-time QR code generation**
- **Supports text, numbers, and URLs**
- **Error animation** for empty input
- **Smooth image reveal animation**
- **Clean and modern UI**

## How It Works

1. The user enters text or a URL in the input field.
2. When the "Generate QR Code" button is clicked:
   - JavaScript checks if input is valid.
   - Sends the input to the QR API.
   - Updates the `img` element with the generated QR image.
3. If no input is entered, an **error shake animation** is triggered.

## Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling & animations  
- **JavaScript (ES6)** – Logic & API handling  
- **QRServer API** – Generates the QR code image
